## 🤔 > 시간 차이가 생긴 이유
- 0번: 저작권에 걸리지 않으면서 통일된 느낌의 이미지르 만들려고 하다보니 예상보다 더 소요되었다.
- 1번: 화면 간 연결을 segue로 하였다가, 코드로 전환하느 방식으로 바꾸게 되어 예상보다 더 소요되었다.
- 2번: 그냥 예상 시간을 너무 짧게 잡았던 것 같다.

- 4번: repository를 수정하는 과정과 viewController들에 주입해주는 방식에 대해 고민하고 수정하느라 예상보다 더 소요되었다.
- 6번: 커피목록에 기본 커피들을 넣어주고 싶은데, 어떤 방식으로 넣어주어야할지, 그리고 기본 목록으로 어떤 커피들이 있으면 좋을지, 몇 개정도가 있으며 적당할지, cell의 높이는 어느정도 인게 UX적으로 좋을지 고민하고 다양하게 테스트해보느라 예상보다 더 소요되었다.
- 8번: UIImagePicker를 사용해서 앱에서 사진첩에 접근할 떄 권한을 묻지 않았다. 이 부분을 몰랐어서 무언가 놓친줄 알고 엄청 고민하였으나, 나중에 업데이트때 Photo kit을 적용하여 사진첩 접근 권한도 요청하도록 하려 한다. (이 과정이 없었으면 예상 시간보다 적게 걸렸을 것 같다.)
