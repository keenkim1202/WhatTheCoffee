# WhatTheCoffee
- 오늘의 커피 추천 및 방문기록 앱

## Library & Framework & API
- IQKeyboardManager
- Naver Maps API
- Kakao Map API
- Realm
- Firebase (Analytics, Crashlytics)
- SwiftyJSON
- Alamofire
- TextFieldEffects

## 1.1 업데이트
1. 근처 카페 탭 추가
- 현재 위치를 기준으로 근처 카페 목록을 보여줍니다.
- 카페의 상세 주소와 정보를 확인할 수 있습니다.

2. 기본 이미지 재추가
- '추천'탭 왼쪽 상단의 설정 화면에서 기본이미지를 재 추가할 수 있습니다.

## 1.2 업데이트
1. 기본 이미지 에셋 추가
- 기본으로 제공하는 따뜻한 음료 에셋이 추가되었습니다.

2. 근처 카페탭 상세페이지 닫기 버튼 추가
- 상세페이지가 모달형식으로 되어있어서 위에서 아래로 스와이프해서 화면을 벗어나야하느 부분에 대해 불편하다는 피드백을 받았습니다.
- 그래서 화며 상단에 닫기 버튼을 추가하였습니다.


## 1.3 업데이트

1. 기본 이미지 재추가 기능 수정
- 사용자의 선호에 따라 아이스음료전체/따뜻한음료전체/개별음료 를 선택하여 재추가할 수 있도록 하였습니다.

2. 버그 수정
- '근처 카페'탭에서 카페 정보 화면의 닫기 버튼을 눌렀을 때, 화면 상단으로 자동 스크롤이 되는 부분을 개선하였습니다.

3. 일부 폰트가 적용되지 않던 화면 수정

4. 버전 정보 아래 부분에 배경색 적용
