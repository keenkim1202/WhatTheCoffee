# WhatTheCoffee
- 오늘의 커피 추천 및 방문기록 앱

## Library & Framework & API
- IQKeyboardManager
- Naver Maps API
- Kakao Map API
- Realm
- Firebase (Analytics, Crashlytics)
- SwiftyJSON
- Alamofire
- TextFieldEffects

## 1.1 업데이트
1. 근처 카페 탭 추가
- 현재 위치를 기준으로 근처 카페 목록을 보여줍니다.
- 카페의 상세 주소와 정보를 확인할 수 있습니다.

2. 기본 이미지 재추가
- '추천'탭 왼쪽 상단의 설정 화면에서 기본이미지를 재 추가할 수 있습니다.
