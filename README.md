# WhatTheCoffee
- 오늘의 커피를 추천하고, 현재 사용자의 위치를 기준으로 가장 가까운 커피점을 추천합니다. 
- 자신이 방문한 카페에 대해 별점과 간단한 코멘트를 남길 수 있습니다.
